## Assessment Types

### Mini-Tasks Created (73 total)
For practical modules with assessments, mini-tasks were created for:
- **Online** - Online assessments
- **MT** - Module Test
- **MT1, MT2** - Multiple module tests
- **IWP** - In-class Written Project
- **CWP** - Class Written Project

### Status Types
- **P/NYP** - Pass/Not Yet Passed (used for Fundamentals, Theory, and Practical modules)
- **C/NYC** - Complete/Not Yet Complete (used for Life Skills and Tooling U modules)

## How to Use

### Recording Module Progress

1. **For modules with mini-tasks (Practical Modules):**
   - Navigate to: Students → Record Attempt → Select Module → Select Mini-Task
   - Record Pass/Fail for each assessment type (Online, MT, IWP, CWP)
   - Add unlimited attempts per assessment type

2. **For modules without mini-tasks (Fundamentals, Tooling U, Theory):**
   - Navigate to: Students → Record Attempt → Select Module → "Record Module Result"
   - Select status based on module type:
     - P/NYP modules: Pass, Fail, or In Progress
     - C/NYC modules: Complete, Not Yet Complete, or In Progress

### Viewing Modules
All modules are now available in the system with:
- Full module names
- Module codes (where applicable)
- Category classification
- Status type indicators
- Credit percentages (where applicable)
- Associated assessment mini-tasks (for practical modules)
