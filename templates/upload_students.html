Excel File Format

Required Columns (at least one):

  • Student Name or student_name - The student's full name

Optional Built-in Columns:

  • Group Name or group_name - Student's group/class
  • Level or level - Student's level/grade
  • Mark or mark - Student's current mark (number)

Custom Columns:

  • Any other columns will be detected and you can choose to add them as custom fields
  • Examples: Gender, Race, Area of Residence, Phone Number, etc.
Note: The first row should contain column headers. Data should start from the second row.
